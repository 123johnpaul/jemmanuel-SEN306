# SEN306 – Software Construction

This repository contains my **coursework, assignments, labs, and projects** for the course **SEN306 – Software Construction**.

## 📘 About the Course

**Software Construction** focuses on the practical aspects of building software systems. The course emphasizes writing clean and maintainable code, applying coding standards, debugging, testing, and using version control during software development.

This repository serves as a **personal archive of my work and learning progress throughout the course**.

## 🎯 Purpose of this Repository

The main purpose of this repository is to:

- Store my **assignments**
- Keep track of **lab exercises**
- Maintain **course projects**
- Practice **software construction techniques**
- Document my **learning progress**
